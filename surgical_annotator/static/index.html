Annotation Guide

Overview

This tool collects two types of training data:

  1. Segmentation masks & keypoints — train a visual detection model (per-frame, in Tools/Pegs mode)
  2. Phase labels — train a temporal phase recognition model (across frames, in Batch mode)

Recommended Workflow

Step 1: Phase Labels First (Batch Mode)

Phase labelling is fastest done in bulk. For each trial:

  1. Press B to open Batch Mode
  2. Scroll through thumbnails to identify phase transitions
  3. Click the first frame of a phase, then Shift+click the last frame to select a range
  4. Press the phase key (Z=Idle, X=Reach, C=Nudge, D=Grasp, R=Transfer, T=Place, G=Dropped)
  5. Set Cycle number and active Tool (T1/T2) for each range
  6. Continue until all frames are labelled, then close Batch Mode

Tip: Do all phase labels for a trial before moving to segmentation. This lets you focus on temporal flow without switching context.

Step 2: Tool Segmentation (Tools Mode)

For each sampled frame in the trial:

  1. Select Tool 1, press 1 for mask — draw the segmentation polygon
  2. Press 2/3 for top/bottom shaft lines
  3. Press 7 for joint, 8 for EE tip, Q/W for EE left/right
  4. If auto-advance is on (Tab), the next tool activates automatically
  5. Repeat for Tool 2 (4–6, 9, 0, O/I)
  6. If a tool is out of view, press Shift+1 or Shift+2 to mark it as Out
  7. Use P to copy annotations from the previous frame, then adjust
  8. Press N to advance to the next frame

Step 3: Peg & Pegboard Segmentation (Pegs Mode)

Switch to Pegs mode (M) for peg transfer board annotations:

  1. Press 1–6 to select a peg; mask drawing auto-starts
  2. After mask, place 3 keypoints (U/Y/J). The next peg auto-selects
  3. Set peg state (Source/Grasp/Transfer/Target) and which post it's on
  4. Annotate source posts (S1–S6) and target posts (T1–T6) with masks + keypoints
  5. Draw the board outline (L)
  6. Use K to copy all peg/pegboard data from the most recent annotated frame

Efficiency Tips

  • Copy Prior is your best friend — annotations change slowly between sampled frames. Copy prior, then nudge vertices in Edit mode
  • SAM assists masks — use "New SAM" to auto-generate tool masks, then refine
  • Batch before detail — label all phases for a trial first, then do per-frame segmentation
  • Skip broken frames — press S to skip frames with camera issues
  • Keyboard over mouse — all tools have keybinds. Learn them to go 3× faster
  • Occluded parts — if a keypoint/tool part is hidden, set its visibility to Occluded (orange status). Don't guess positions
